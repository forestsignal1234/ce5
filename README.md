# CE-5 at The Experiencer Center

A landing page for a CE-5 contact group hosting peaceful, human-initiated
night gatherings under the dark skies of eastern Washington State.

The site covers what CE-5 is, the season's upcoming gatherings (each with its
own detail page), the retreat venue and amenities, and how to get involved.

## Files

| Path | What it is |
|------|------------|
| `index.html` | **Hostable build.** A single self-contained file with everything inlined — open it directly in a browser or deploy it as-is. |
| `CE-5 Retreat.dc.html` | The editable source design. References `support.js` and `assets/`. |
| `support.js` | Runtime that powers the source design file. |
| `assets/` | Images used by the page (hero photo). |

## View it locally

Just open `index.html` in any browser — no build step, no server required.

To work on the editable source instead, keep `CE-5 Retreat.dc.html`,
`support.js`, and `assets/` together and open the `.dc.html` file.

## Deploy with GitHub Pages

1. Push this repo to GitHub (see steps below).
2. In the repo, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to *Deploy from a branch*,
   pick the `main` branch and the `/ (root)` folder, then **Save**.
4. After a minute your site is live at
   `https://<your-username>.github.io/<repo-name>/`.

GitHub Pages serves `index.html` automatically.
